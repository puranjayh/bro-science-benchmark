# BroScienceBench

A 246-item benchmark for strength-training misinformation in LLMs.

- data/ — benchmark (v1.1, 246 items)
- scripts/ — generators, eval harness, analysis
- results/ — raw model outputs + stats
- DATASHEET.md — datasheet

Set your own OpenRouter key in .env.local (see .env.example). Scripts use relative paths from the original layout; adjust as needed.
